# Novite

Novite is an investigation canvas for mapping people, evidence, events, and concepts on an infinite board.

## Run

```bash
bun start
```

## Features

- Infinite pan + zoom canvas
- Right-click context actions for board, nodes, and links
- Multi-select and group drag for reorganization
- Image upload and screenshot preview on nodes
- Autosave to MongoDB
